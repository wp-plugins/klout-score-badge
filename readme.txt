== Description ==
Klout Score is a measure of your social media influence .This simple plugin displays your Klout profile and score. For more details visit http://www.techlineinfo.com/wordpress-plugin-to-display-your-klout-score/
 = 1.0 = 
Release version

1.Upload the  plugin to your blog and just activate it.
2.Goto your Appearance>Widgets and then add your klout id. 

That's all !!!
